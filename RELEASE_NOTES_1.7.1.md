# pfSense Manager 1.7.1

## Highlights

- Restored live combined and per-interface Network Monitor traffic charts.
- Improved Network Monitor chart readability with inbound traffic above zero and outbound traffic below zero.
- Added adaptive, human-friendly bandwidth scaling that follows visible traffic levels.
- Added a saved display-unit selector for bits per second or bytes per second.
- Added readable bandwidth-axis labels, time labels, legends, fills, and tooltips.
- Added an in-app notification when a newer GitHub release is available.
- Preserved session-safe polling, stale-response protection, lifecycle handling, and saved refresh settings.

## Version

- App version: `1.7.1+10`
- Android application ID: `com.privacyog.pfsense_manager`

## Upgrade note

Users already running version 1.7.0 with the current application ID can install 1.7.1 as a normal update, provided the APK is signed with the same release key.
